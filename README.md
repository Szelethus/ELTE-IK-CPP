# C++ gyakorlati jegyzetek

Maga a compiled pdf az bátran bárkivel megosztható ;)

## Pótlandó dolgok

* Implicit konverziók
* user defined converson
* using namespace std;
* UML diagramok 11 gyakhoz
* Kiértékelési sorrend
* Nyelvi elemek
* istream/ostream iterator
* jobb és balérték
